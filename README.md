# Hotel Booking Cancellation Analysis

### Business Problem
Online Travel Agencies face 37% average booking cancellation rates, directly impacting partner hotel revenue and platform commissions.

My Approach
Analyzed 119,390 hotel bookings using Python (Pandas, Matplotlib) to identify seasonal cancellation drivers.

Key Findings
June has 41.46% cancel rate vs **37% average**
August loses 5,239 bookings - highest volume month
Peak season months show 40%+ cancellation rates
Business Recommendation
Implement targeted retention strategies for summer months. A 5% reduction in June cancellations alone recovers 547 bookings.

Tools Used
Python, Pandas, Jupyter, Matplotlib, Seaborn
